VideoForge FAQ

Branded search answers for VideoForge / Video Forge Studio.

Is VideoForge Windows-only?

Yes. The public download is a Windows 10/11 installer. Hosted or Docker paths exist in the repo for operators, but this page sells the free desktop download.

Is there a subscription to download?

No. Install is free. Optional credit billing for add-ons uses the same Knight Account Stripe path as PixelForge when enabled — not required to render local ranked exports.

What does “ranked local video” mean?

You upload 3–10 clips, set titles/captions/overlays, and render a countdown or ranked showcase with FFmpeg on your machine — not a YouTube Shorts auto-reframe tool.
